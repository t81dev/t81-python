# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-02-08

### Added

- Initial typed package scaffold with ternary core and quantization helpers.
- Optional integrations for Hugging Face and llama.cpp.
- End-to-end checkpoint export pipeline (`export-hf-json`, `export-hf`).
- Artifact inspection and validation CLI command (`inspect-artifact`).
- CI and publish workflows for quality checks and tag-based release.
- Example scripts, benchmark tooling, and expanded documentation.
